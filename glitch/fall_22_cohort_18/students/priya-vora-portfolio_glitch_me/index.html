Recent Projects
Python Project - Assembly Language with Raspberry Pi

This Python Blinking Led project demonstrates the usage and understanding of assembly language that implements a symbolic representation of machine code that is needed for a Raspberry Pi to turn an led light on and off.

The python code programmed reads a set of assembly instructions and converts each of them into binary code. Once done this program encodes and writes to an img type file which then can be saved on an memory chip to use on an acutally Raspberry Pi. The Raspberry Pi is connected to a breadboard to help demonstrate this program.

Challenges
  • Understanding how registers work inside a CPU
  • Comprehending what set of assembly instructions are needed and why

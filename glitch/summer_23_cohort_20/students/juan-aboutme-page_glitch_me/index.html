My Projects

This is my first project. It is a website that helps you find the best gyms in San Francisco. I used Airtable, a cloud-based database service, to store and manage the data of the gyms, such as their name, location, rating, price, and amenities. I used JavaScript and jQuery to fetch the data from Airtable and display it on my website. You can see the website by clicking on this link: My Gym Locator.

This is my second project. It is an IoT project where I used Arduino, a breadboard, and leds to create a password lock. I used a keypad to enter a four-digit password and leds to show the status of the lock. I also used a buzzer to make sounds when the password is correct or incorrect. You can see the project by clicking on this link: My Password Lock.
